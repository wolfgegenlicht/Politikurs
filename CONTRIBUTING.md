# Contributing to PolitiKurs

Thank you for your interest in contributing to PolitiKurs! We welcome contributions from everyone.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/wolfgegenlicht/Politikurs.git
    cd Politikurs
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Set up environment variables**:
    - Copy `.env.example` to `.env.local`.
    - Fill in the required API keys (Supabase, OpenRouter, etc.).
5.  **Run the development server**:
    ```bash
    npm run dev
    ```

## How to Contribute

- **Report Bugs**: Open an issue on GitHub describing the bug and how to reproduce it.
- **Suggest Features**: Open an issue to discuss new ideas.
- **Submit Pull Requests**:
    1.  Create a new branch for your feature or fix.
    2.  Commit your changes with clear messages.
    3.  Push your branch to your fork.
    4.  Open a Pull Request against the `main` branch.

## Code Style

- We use **TypeScript** and **Tailwind CSS**.
- Please ensure your code works and follow the existing style.
